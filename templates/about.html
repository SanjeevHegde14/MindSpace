About MindSpace

MindSpace is an advanced AI-powered platform designed to analyze public social media activity from platforms like Reddit, Instagram, and Twitter/X. By combining Natural Language Processing (NLP), sentiment analysis, and machine learning, MindSpace identifies emotional patterns, behavioural indicators, and potential mental-health concerns based on the content people share online.

Project Objectives

  • Analyze public posts from multiple social media platforms.
  • Identify mood patterns using sentiment analysis.
  • Predict abnormal or concerning behavioural tendencies using machine learning.
  • Provide detailed summaries, graphs, and post-by-post insights.
  • Allow users to save analysis history and download PDF reports.

System Workflow

  1. User Input — The user selects a platform and enters a username or profile link.
  2. Data Scraping — MindSpace retrieves the user’s public posts:
    • Reddit: via asyncpraw API
    • Instagram: via Apify Scraper
    • Twitter/X: via Playwright browser automation
  3. Sentiment Processing — Each post is analyzed using VADER to determine positive, negative, and neutral sentiment scores.
  4. ML-Based Detection — A trained Naive Bayes model estimates the probability of abnormal behavioural expression in posts.
  5. Visualization — The system generates trend graphs, average scores, and a final verdict based on the analysis.
  6. History & Reports — Logged-in users can store analyses and download professionally formatted PDF reports.

Key Features

  • Multi-platform social media analysis.
  • AI-driven sentiment and behavioural scoring.
  • Real-time scraping of public posts.
  • Personal dashboard with saved analysis history.
  • PDF report generation with detailed post-level breakdowns.

Technologies Used

  • Backend: Flask, Python, SQLAlchemy, SQLite.
  • Scraping: asyncpraw (Reddit), Apify API (Instagram), Playwright (Twitter/X).
  • AI/NLP: Naive Bayes model, VADER sentiment analyzer.
  • Frontend: Bootstrap, HTML5, CSS3, Font Awesome.
  • Other: FPDF for PDF reports, dotenv for secure credentials.

Future Scope

  • Integration with additional platforms (YouTube, TikTok, Facebook).
  • Use of transformer-based models (BERT, RoBERTa) for improved accuracy.
  • Long-term emotional trend dashboards.
  • Personalized content recommendations based on mood.
  • Mental-health supportive resources and early-warning mechanisms.

Conclusion

MindSpace bridges the gap between artificial intelligence and mental health awareness. By deeply analyzing how people express themselves online, it offers meaningful, data-driven insights into emotional well-being. With its powerful features, modern UI, and real-time scraping capabilities, MindSpace stands as a valuable tool for researchers, psychologists, developers, and anyone seeking better understanding of digital behaviour.
